# Quiz Game (Python)

An interactive quiz game developed using Python with a graphical user interface (GUI) built using Tkinter. The application provides instant feedback using Turtle graphics with emoji-based visual responses.

---

## 🔹 Features

- Multiple-choice quiz system
- Graphical user interface using Tkinter
- Instant feedback for correct and incorrect answers
- Emoji-based visual representation using Turtle graphics
- Final score display at the end of the quiz

---

## 🔹 Technologies Used

- Python
- Tkinter (GUI)
- Turtle Graphics

---

## 🔹 Application Flow

1. The quiz starts with a question displayed on the screen
2. User selects an answer from multiple options
3. On submission:
   - Correct answer → Smiley face displayed using Turtle
   - Incorrect answer → Sad face displayed using Turtle
4. The next question is displayed automatically
5. At the end, the final score is shown

---

## 🔹 How to Run

1. Install Python (version 3.x)
2. Run the program:

   python quiz_game.py

3. The quiz window will open

---

## 🔹 Output

- Interactive quiz window
- Pop-up messages for correct/incorrect answers
- Turtle graphics window displaying emoji feedback
- Final score displayed at the end
